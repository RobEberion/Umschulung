Fetch API mit JSON: GET und POST

Unterschied zwischen GET und POST

GET-Methode:

  • Wird verwendet, um Daten vom Server anzufordern.
  • Daten werden in der URL (Query-Parameter) gesendet.
  • Für reine Datenabfragen geeignet (z. B. Laden von Inhalten).



POST-Methode:


    
    
  • Wird verwendet, um Daten an den Server zu senden (z. B. Formulare).
    • 
    
  • Daten werden im Body der Anfrage gesendet.
    • 
    
  • Für Änderungen oder das Erstellen von Daten geeignet.
